# Customer Churn Analysis – Power BI Case Study (Databel Telecom)

## Overview
This project analyzes customer churn for a fictional telecom company, **Databel**, using Power BI.  
The goal is not only to measure churn rate but also to understand the key drivers behind customer attrition and identify actionable insights to reduce churn.

## Objective
- Measure overall churn rate
- Identify patterns and trends in customer churn
- Analyze churn across different customer segments
- Build interactive dashboards for business insights
- Support data-driven decision-making for retention strategies

## Dataset
- Source: Databel Telecom (sample case study dataset from DataCamp)
- Type: Subscription-based customer data
- Key fields include:
  - Customer demographics (age, group, etc.)
  - Account details (contract type, account length, payment method)
  - Usage and service-related attributes
  - Churn status

## Tools Used
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Measures & Calculated Columns)
- Data Modeling

## Data Preparation
- Handled missing and inconsistent values
- Created calculated columns for churn classification
- Built measures for:
  - Total customers
  - Churn rate
  - Average monthly charges
  - Segment-wise churn analysis

## Dashboard Features
The Power BI report includes:

- **KPI Overview Page**
  - Total customers
  - Churn rate
  - Average account length
  - Average monthly charges

- **Customer Analysis Page**
  - Churn rate by age group
  - Churn distribution across customer segments

- **Service & Behavior Analysis Page**
  - Customer service calls impact
  - Payment method vs churn
  - Contract type vs churn

- **Interactive Filters**
  - Age group slicer
  - Account length slicer
  - Contract type filter
  - Payment method filter

## Key Insights (Example)
- Certain contract types show significantly higher churn rates
- Customers with higher service call frequency tend to churn more
- Monthly charges influence churn behavior across segments
- Specific customer groups are more retention-sensitive than others

## Project Outcome
This project helped in:
- Strengthening DAX and data modeling skills
- Understanding churn behavior in subscription-based businesses
- Building interactive and insight-driven Power BI dashboards
